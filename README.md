# example-voting
Приложение с голосованием из книги Pro Go<br>

При запуске программы будут доступны ссылки:<br>
```/``` - Главная<br>
```/list``` - Список всех заявок<br>
```/form``` - Отправка заявки. После успешной отправки, откроется страница благодарности<br>
